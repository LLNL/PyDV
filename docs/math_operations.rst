.. _math_operations:

Math Operations
===============

These functions provide a mechanism for performing mathematical operations on specified **plotted** curves
which can be seen with the command `list`.

.. note::
   **< >** = Required user input.

   **[ ]** = Optional user input.

   **[PyDV]:** = Python Data Visualizer command-line prompt.
